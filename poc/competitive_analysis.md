# ToeicMaster 競合分析 (2026-03-07)

## 主要競合

### 1. Santa アルク (santa.alc.co.jp)
- **強み**: AIスコア診断（95%精度、3分12問）、弱点分析→個別最適カリキュラム、50問ごとにスコア更新
- **弱み**: Part1写真問題の画像が限定的
- **価格**: フリーミアム（スコア診断無料、学習は有料）
- **パクるべき**: AIスコア診断、弱点特定、予測スコア自動更新

### 2. abceed (abceed.com)
- **強み**: 20,000問超のAIレコメンド、忘却曲線ベースの復習、70%正答率に調整された難易度
- **弱み**: 有料版でないとAIレコメンド使えない
- **価格**: フリーミアム（AIレコメンドは有料）
- **パクるべき**: 忘却曲線の復習タイミング、適応型難易度調整

### 3. TOEIC Test Pro (estudyme.com)
- **強み**: 登録不要、ダークモード、フル模試(120分)+ミニテスト(60分)、AI採点(S&W)
- **弱み**: Part1問題の質は普通
- **価格**: 無料
- **パクるべき**: 登録不要で即開始、ダークモード、ミニテストモード

### 4. toeic.kakomonn.com
- **強み**: 完全無料、シンプルUI、解説付き
- **弱み**: 問題数限定、古い
- **パクるべき**: シンプルさ、「続きから出題」機能

## ToeicMasterの差別化ポイント

| 競合の弱点 | ToeicMasterの強み |
|-----------|-----------------|
| Part1写真が少ない・古い | フリー画像400万枚+からAI自動生成 → 無限の問題 |
| AI生成画像が不自然 | Unsplash/Pexels のリアル写真 → 本番に近い |
| 問題やり尽くす | 毎回異なる問題を生成可能 |
| 単一LLMの回答 | マルチエージェントで画像分析+問題生成+品質チェック |

## 採用する競合機能（優先順）

1. **登録不要で即開始** (TOEIC Test Pro) — 摩擦を最小化
2. **AIスコア診断** (Santa) — 3分でスコア予測
3. **適応型難易度** (abceed) — 70%正答率ターゲット
4. **忘却曲線復習** (abceed) — 間違えた問題を最適タイミングで再出題
5. **ダークモード** (TOEIC Test Pro) — 基本UX
6. **ミニテストモード** — 5問/10問の短時間練習
7. **進捗ダッシュボード** — スコア推移グラフ
